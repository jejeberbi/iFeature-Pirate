iFeature-Pirate pour Wordpress
===============

iFeature est un thème pour wordpress, diffusé sous licence GNU GPL V2.

Ce thème a été modifié afin d'être aux couleurs du Parti Pirate ainsi que de la région Franche-Comté car ce thème est travaillé pour le futur site de la futur section locale.

Le thème propose par défaut deux nuances de couleurs : gris et vert ; qui sont décrit dans css/colors. Le CSS modifié est le green.css (celui qui n'est pas séléctionné automatiquement lors de l'installation du thème).
Des modifications ont aussi été faites sur les feuilles de style globales : css/style.css notamment.


Toutes les MAJ qui seront faites sur ce dépôt seront aussi appliqué sur la maquette actuelle : http://jejeberbi.legtux.org/wordpress/

Installation du thème
==============

Si vous souhaitez installer le thème en local, récupérez le contenu du dépôt, placez le à l'intérieur d'un dossier que vous nommerez comme vous voulez. Ensuite, placez ce dossier dans wp-content/themes de votre installation wordpress.
Il ne vous reste plus qu'à activer le thème via le tableau de bord de wordpress.
